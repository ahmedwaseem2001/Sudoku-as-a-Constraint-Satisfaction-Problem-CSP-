# Sudoku-as-a-Constraint-Satisfaction-Problem-CSP-
Sudoku Solver using Constraint Satisfaction Problem (CSP) in AI. Solves 9×9 puzzles using Backtracking Search with AC-3, Forward Checking, and MRV heuristic for efficient solving of easy to hard puzzles. Reads puzzles from .txt files and displays solution with performance stats (calls &amp; failures).
